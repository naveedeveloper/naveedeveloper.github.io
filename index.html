Property Value Estimator

    Github Repo: https://github.com/naveedeveloper/property-value-estimator
    The project is divided into three components:
    1. ML Model:
    • Firstly built a predictive model using Scikit-Learn's Linear Regression.
    • The model was trained on the Bangalore home prices dataset obtained from Kaggle.
    • The dataset was used to train the model to predict home prices based on features such as square feet, the number of bedrooms, and the number of baths.
    2. Python Flask Server:
    • The second step involved creating a Python Flask server.
    • This server utilized the saved machine learning model to serve HTTP requests.
    • When a client made an HTTP request with features like square feet, location, the number of baths, and the number of bedrooms, the Flask server used the model to predict the home price and returned the result as a response.
    3. Web Application (HTML, CSS, JavaScript):
    • The third component of this project was a web application built using HTML, CSS, and JavaScript.
    • Users were able to interact with the model through this web interface.
    • They could input property details such as square footage and the number of bedrooms, and the web application made requests to the Python Flask server to retrieve the predicted home price.

Car Price Prediction

    Github Repo: https://github.com/naveedeveloper/car-price-predictor
    The project can be divided into the following key steps:
    1. Data Cleaning and Feature Engineering:
    • In the initial phase, the dataset was thoroughly cleaned and preprocessed to ensure data quality.
    • After data cleaning, feature engineering was performed to extract meaningful insights and create relevant features for the prediction task.
    2. Model Building:
    • A Linear Regression model was constructed to predict car prices based on the processed data.
    • The model achieved an R2 score of 0.92 on the test data, demonstrating its predictive capabilities.
    3. Flask Web Application:
    • To make the model accessible and user-friendly, a web application was developed using Flask.
    • Users can input car details, and the application will provide price predictions based on the model's calculations.
